# Installation options

## Online install (recommended)

- Run the installer directly from an Arch-based live environment.
- Pulls the latest repo state at install time.
- Requires a working network connection.
- Fastest setup with the least prep.

Best when:

- You want the latest defaults.
- You can install from a live USB with internet access.

## Build ISO

- Build your own Arch-based live ISO with `archiso`.
- Include the package lists from this repo in your profile.
- Run the installer after booting the ISO.
- Linux is required to build the ISO (native or in a Linux VM).

Best when:

- You want a repeatable, prebuilt installer image.
- You need to customize the live environment before install.
